# loan-risk-sql-analysis
# Loan Risk and Lending Analysis using SQL

This project analyses consumer loan data from a US bank to identify borrowing patterns, credit risk and lending trends.

The analysis was made using SQL to extract and summarise key information from the dataset, and Power BI to visualise insights and support business decision making.

## Tools
- SQL
- Power BI
- Data Analysis

## Project Objectives
- Analyse loan demand and customer borrowing behaviour
- Identify credit risk indicators
- Support better lending decisions using data insights

## Key Analysis
The SQL queries were used to explore several business questions:

- Loan purpose distribution
- Loan term preferences (36 vs 60 months)
- Interest rate distribution
- Geographic loan concentration
- Loan grade risk levels
- Loan repayment status

## Key Insights
- Home improvement loans represented the highest demand category in 2015.
- Most borrowers preferred 36-month loans over longer terms.
- Interest rates ranged from 5.31% to 28.49%, reflecting different risk levels.
- Lending activity was concentrated in California, New York and Texas.
- Most loans were classified as “Current”, indicating strong repayment behaviour.

## Business Impact
The findings help financial institutions understand borrower behaviour, monitor credit risk and optimise lending strategies while maintaining portfolio stability.
